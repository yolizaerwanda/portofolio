## Daftar Project

### 1. SIGAP - Sistem Pengaduan Online Polisi Pauh
Role: Project Manager  
Tahun: 2025  

#### Deskripsi
SIGAP adalah sistem berbasis web yang dirancang untuk mempermudah masyarakat dalam mengajukan pengaduan secara online ke kantor polisi Pauh. Sistem ini membantu mempercepat proses pelaporan kejadian serta meningkatkan efisiensi pelayanan publik.

#### Fitur Utama
- Pengajuan laporan/pengaduan secara online  
- Manajemen data laporan oleh admin  
- Tracking status laporan  
- Dashboard monitoring  

#### Teknologi
- Laravel  
- MySQL  
- Git & GitHub  

#### Tanggung Jawab
- Memimpin tim dalam perancangan dan pengembangan sistem  
- Mengatur alur kerja tim  
- Mengawasi proses development hingga deployment

### 2. Sistem Web Retur PT. Glorienta Panca Henna
Role: Fullstack Developer  
Tahun: 2025  

#### Deskripsi
Sistem informasi berbasis web yang digunakan untuk mengelola proses retur barang antara admin dan sales di PT. Glorienta Panca Henna. Sistem ini membantu pencatatan dan pengolahan data retur secara lebih terstruktur.

#### Fitur Utama
- CRUD data retur barang  
- Manajemen data sales dan admin  
- Tracking status retur  
- Laporan data retur  

#### Teknologi
- Next.js  
- MySQL  
- Git & GitHub  

#### Tanggung Jawab
- Mengembangkan frontend dan backend sistem  
- Mendesain database  
- Mengimplementasikan fitur CRUD

### 3. SHOP YOLIZA - Mobile E-Commerce App
Role: Fullstack Developer  
Tahun: 2025  

#### Deskripsi
Aplikasi mobile e-commerce yang menyediakan berbagai kategori produk seperti elektronik, pakaian pria dan wanita, serta sepatu pria dan wanita. Aplikasi ini dirancang untuk memberikan pengalaman belanja yang mudah dan efisien bagi pengguna.

#### Fitur Utama
- Splashscreen  
- Login dan autentikasi pengguna  
- Pencarian produk  
- Katalog produk  
- Keranjang belanja  
- Sistem pembayaran  

#### Teknologi
- Flutter  
- Git & GitHub  
- Cloudflare  

#### Tanggung Jawab
- Mengembangkan aplikasi mobile dan integrasi backend  
- Mendesain antarmuka pengguna (UI/UX)  
- Mengimplementasikan fitur e-commerce
